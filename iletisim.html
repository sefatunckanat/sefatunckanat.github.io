Hakkımda
Indie oyun geliştirme ile başladığım geliştirici macerası, lisede Veri Tabanı Programcılığı okumasıyla taçlandı. Derslerde C# ve SQL gördü. Oyun geliştirmeye Game Maker ile başlayıp Unity ile devam etti. Bunun yanında Open Source oyun motorlarını kullanmak için Temel düzeyde JAVA öğrendi. Bildikleri onu mobil platformla yöneltti. Native Android öğrenip ufak projeler geliştirdi. Üniversitede gördükleri bildiklerinin basit tekrarı olsada üniversite bitişiyle birlikte Mobil veya Web tabanlı uygulama ve oyun geliştirecek düzeye geldi.

HTML + CSS + JS üçlemesiyle yaptığı tasarımları PHP yürütüyor. PHP ile yazdığı web servisleriyle mobil uygulamalar geliştirmekte. Her boşluktada yeni oyun ve uygulama fikirlerini hayata geçirmeye çalışıyor.
